# RV001 — Image-First Remote Viewing

## Purpose
Test whether an F12 shell improves image-first capture and later blind judging against hidden targets.

---

## Protocol
Use `RV001` from `psi-protocols.md`.

Capture in this order:
- shapes
- textures
- temperature / motion feel
- colours
- atmosphere
- sketch
- only then interpretive guess if desired

---

## Conditions
Suggested first comparison:
1. neutral bed
2. F10 shell
3. F12 shell

---

## Minimum Run Plan
Suggested first block:
- 5 sessions neutral bed
- 5 sessions F10
- 5 sessions F12

Then review before extending.

---

## Judging
Primary:
- rank-order judging against decoys

Secondary:
- descriptor overlap grid

---

## Log Required
- condition
- confidence
- imagery density
- sketch produced? yes / no
- rank score
- descriptor score

---

## Watch For
- naming objects too early
- retrofitting vague descriptors after feedback
- scoring while still emotionally attached to the session

---

## Success Question
Does the shell increase:
- image richness?
- structural accuracy?
- judgeability?
without increasing noisy fantasy material too much?
